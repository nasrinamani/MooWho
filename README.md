# Moo Who?
![MooWho Gameplay](screenshot.png)
**Moo Who?** is an educational and fun hide-and-seek game designed for children. The game is designed for young children who are not yet able to read, challenging them to recognize animal sounds and find the matching animals. Itâ€™s a simple, playful way to test listening and identification skills through sound.

## Features
- Clickable animals that play realistic sounds
- Find hidden animals based on sounds
- Background music and engaging visuals
- Simple and intuitive interface suitable for kids
- Developed in C++ using **OpenGL, GLFW, GLEW, OpenAL, and FreeGLUT**

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/nasrinamani/MooWho.git
```
2. Install the required libraries: OpenGL, GLFW, GLEW, OpenAL, FreeGLUT
3. Compile and run the game using your preferred C++ IDE
4. Explore and find animals to unlock their sounds!
